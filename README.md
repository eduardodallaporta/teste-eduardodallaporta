# 📊 Teste Técnico

Este repositório contém as soluções desenvolvidas para o teste técnico.

---

## 📁 Estrutura do Projeto

O projeto está organizado em quatro módulos:

- **01_SQL** → Scripts de criação e consultas para análise de vendas  
- **02_Analise_Python** → Notebook com leitura e análise dos dados  
- **03_Aplicacao_WEB** → Aplicação web para visualização das vendas  
- **04_PowerBI** → Dashboard interativo com indicadores de vendas  

Cada módulo possui um arquivo README.md com instruções específicas de execução e descrição das funcionalidades.
