# Módulo de Visualização - Power BI

Este diretório contém o dashboard desenvolvido no Power BI.

## Arquivo
- `Visualizacao_Grafico.pbix`

## Conteúdo do Dashboard
- KPI: Total de Vendas
- KPI: Quantidade de Pedidos
- Gráfico de Receita por Produto
- Segmentação de dados por Status do Pedido

## Funcionalidades
A seleção de um Status no slicer atualiza automaticamente:
- O valor dos KPIs
- O gráfico de Receita por Produto
