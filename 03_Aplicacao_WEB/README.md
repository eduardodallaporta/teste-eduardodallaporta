# Módulo Web - Gestor de Vendas

Tarefa 03 - Aplicação Web

## 🌐 Visualização Online

A aplicação está disponível para acesso direto via GitHub Pages:

🔗 https://eduardodallaporta.github.io/teste-eduardodallaporta/

---

## 📁 Arquivos

- `index.html`: estrutura da página web
- `style.css`: estilização da interface
- `script.js`: lógica de leitura e exibição dos dados
- `vendas.json`: base de dados consumida pela aplicação

---

## ⚙ Funcionalidades

- Exibição dinâmica das vendas em tabela HTML
- Leitura dos dados a partir do arquivo `vendas.json`
- Destaque visual para pedidos com status **Cancelado**
- Botão de visualização de detalhes por pedido

---

## ▶ Execução Local (Opcional)

Caso deseje executar a aplicação localmente:

- Recomenda-se utilizar a extensão **Live Server** no Visual Studio Code.

Abrir o arquivo diretamente via `file://` pode impedir o funcionamento do método `fetch()` devido às políticas de segurança do navegador no carregamento de arquivos locais.
